东北林业大学教务处管理系统
==========================

该应用采用WebView来实现，提供用户访问东北林业大学教务管理系统。简单的说如同一个定向了的浏览器。<br/>
[应用下载](https://github.com/david-loman/WebAppliction/blob/master/jwcglxt_nefu.apk) <br/>
ps：进入上述页面后，点击Raw就能下载到电脑，然后怎么办就看你的了。
###使用说明
=====================

进入应用后直接就可以使用了。
####基础教程
=======================

1 退出程序：<br/>
通过菜单中的退出选项可以退出应用，在登陆系统后按返回键将会报一定错误。<br/>
<br/>
2 更换账户：<br/>
通过系统的退出选项可以回到重新登陆界面，更换账户并登陆即可，<br/>
<br/>
3 设置选项：<br/>
设置选项目前仍未完成，有网页跳转，屏幕切换等内容，敬请期待吧。<br/>
<br/>
4 切换到新系统 <br/>
假如你在使用旧系统，而后想要切换到新系统，可以通过两钟方式实现：<br/>
  (1)页面登陆：在菜单栏中找到“新系统”选项，单击它就可以了。<br/>
  (2)直接登陆：在菜单栏中找到“切换用户”选项，单击它，在跳出的对话框中输入你的学号，密码，点击确定就可以了。<br/>
以上方法同样适用于你处于新系统的时候。<br/>
5 切换到旧系统<br/>
在菜单中选择旧系统，就可以了。<br/>
6 修改默认登陆账户<br/>
很不幸目前尚不支持，如果想修改的话，可以清空程序数据，然后重新填写设置就可以修改了。<br/>
####注意事项：
======================

<hr>0.1.1</hr>
 （1） 在登陆的过程中，可能会因为验证码错误而回到登陆界面
 （2） 登陆时使用Cookie功能可能会出现无法登陆的错误，重新填写信息就行<br/>
 （3） 查询成绩课表时，需要将下拉的菜单重新打开才能查询其他功能<br/>
 （4） 双击可以放大屏幕，缩小也是双击；所以用双手就可以了<br/>
<hr>0.1.2</hr>
  (1)  不按常理操作将会导致应用关闭。不按常理的意思是：按了不该按的确定键。
  (2)  菜单栏中的设置功能只能看，不能用，想修改默认登陆的系统和账户，请自行在系统的应用程序管理中，将该应用的数据清空。
  (3)  第一次登陆，请看好注意中提示的内容。如果选择新教务系统，请输入正确的学号和密码，否则由于个人比较懒，更改很麻烦，方法如上。 
  (4)  推荐使用新教务系统，因为我比较懒。。。不喜欢每次填学号，密码，验证码什么的。。。  
  (5)  如果出现在新系统下打开程序时，显示需要在另一窗口登陆，可通过注意中的(2)来解决。当然，如果出现了这样的情况，我希望您能将使用的情况反馈给我，这样我就能够做出调整。
