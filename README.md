东北林业大学教务处管理系统
==========================

该应用采用WebView来实现，提供用户访问东北林业大学教务管理系统。简单的说如同一个定向了的浏览器。
[应用下载](https://github.com/david-loman/WebAppliction/blob/master/jwcglxt_nefu.apk) <br/>
ps：进入上述页面后，点击Raw就能下载到电脑，然后怎么办就看你的了。
###使用说明
=====================

进入应用后直接就可以使用了。
####基础教程
=======================

1 退出程序：<br/>
通过菜单中的退出选项可以退出应用，在登陆系统后按返回键将会报一定错误。<br/>
<br/>
2 更换账户：<br/>
通过系统的退出选项可以回到重新登陆界面，更换账户并登陆即可，<br/>
<br/>
3 设置选项：<br/>
设置选项目前仍未完成，有网页跳转，屏幕切换等内容，敬请期待吧。<br/>
<br/>
4 注意事项：<br/>
 （1） 在登陆的过程中，可能会因为验证码错误而回到登陆界面
 （2） 登陆时使用Cookie功能可能会出现无法登陆的错误，重新填写信息就行<br/>
 （3） 查询成绩课表时，需要将下拉的菜单重新打开才能查询其他功能<br/>
 （4） 双击可以放大屏幕，缩小也是双击；所以用双手就可以了<br/>
